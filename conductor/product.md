# Initial Concept
Stock Journal은 개인 투자자를 위한 현대적인 주식 매매 일지 애플리케이션입니다. 매매 기록, 수익률 분석, 리스크 관리를 한 곳에서 관리하며, 데이터 기반의 투자 결정을 돕습니다.

# Product Definition - Stock Journal

## 2. Target Audience
- 장기 성장 및 가치 투자에 집중하는 개인 투자자
- 빈번한 기록과 복기가 필요한 데이트레이더 및 스윙 트레이더

## 3. Core Values & Features
### Core Values
- **실시간성**: 주가 데이터 연동을 통한 정확한 수익 추적
- **분석성**: AI 기반의 매매 분석 및 전략 고도화 지원
- **접근성**: 모바일 환경에서도 간편하게 기록할 수 있는 반응형 디자인

### Key Features
- **실시간 데이터 통합**: 국내(KOSPI/KOSDAQ) 및 미국(NYSE/NASDAQ) 시장 데이터 연동
- **AI 기반 매매 분석**: 투자 패턴 분석 및 전략 추천 기능
- **이동성 최적화**: 모바일 기기에 최적화된 매매 기록 및 조회 인터페이스

## 4. Brand & Design Direction
- **Modern & High-Tech**: 핀테크 스타트업 느낌의 세련되고 혁신적인 이미지
- **Minimalist**: 불필요한 요소를 배제하고 데이터와 지표에 집중할 수 있는 깔끔한 UI
