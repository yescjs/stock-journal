# Tech Stack - Stock Journal

## 1. Core Frameworks & Languages
- **Language**: TypeScript (v5.0+)
- **Framework**: Next.js (v16.0+, App Router)
- **Runtime**: Node.js

## 2. Frontend Development
- **UI Library**: React (v19.0+)
- **Styling**: Tailwind CSS (v4.0+)
- **Typography**: Pretendard (Primary), JetBrains Mono (Code/Numbers)
- **Animations**: Framer Motion (Page Transitions, Bottom Sheets)
- **Icons**: Lucide React
- **Data Visualization**: Recharts

## 3. Backend & Data
- **Backend-as-a-Service**: Supabase
- **Database**: PostgreSQL (via Supabase)
- **Authentication**: Supabase Auth
- **ORM/Client**: Supabase JS SDK

## 4. Quality Assurance & Tools
- **Testing**: Playwright
- **Linting**: ESLint
- **Type Checking**: TypeScript
- **Task Management**: Conductor
